# Disconnect Event

## Description

Disconnects an event by its EventConnection output.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### EventConnection (EventConnection)

The event connection that will be disconnected.

```
MouseClickEvent
```
