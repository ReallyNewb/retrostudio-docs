# Collision Group Set Collidable

## Description

Changes collision status between two groups.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Object (Object)

The object whose ancestry (full path, for example Game.Workspace.Object) will be output.

* [x] Can use Variable

```
game.Workspace.Part
```

## Outputs

#### FullName (String)

The ancestry of the object.

```
Game.Workspace.Part
```
