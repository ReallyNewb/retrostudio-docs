# Set Leaderstat

## Description

Sets the value of a leaderstat.

#### Available to

* [x] Scripts
* [ ] LocalScripts
* [ ] Yields

## Inputs

#### Player (Object)

The player whose leaderstat will be changed.

* [x] Can use Variable

```
game.Players.robloxandtyler
```

#### StatName (String)

The leaderstat's name that will be changed for the player.

* [x] Can use Variable

```
GameWins
```

#### Value (Number)

The new value of the leaderstat.

* [x] Can use Variable

```
100
```
