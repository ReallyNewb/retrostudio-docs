# Is Player Banned

## Description

Checks if a player is banned or not.

#### Available to

* [x] Scripts
* [ ] LocalScripts
* [ ] Yields

## Inputs

#### UserId (Number)

The UserId, to check their ban status.

* [x] Can use Variable

```
1
```

## Outputs

#### Result (Bool)

Determines whether or not the UserId has been banned.

```
True
```
