# Get User ID From Name

## Description

Gets a Roblox UserId from a Roblox username.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [x] Yields

## Inputs

#### Username (String)

The username whose user ID will be retrieved.

* [x] Can use Variable

```
ROBLOX
```

## Outputs

#### UserId (Number)

The UserId that belongs to the provided Roblox username.

```
1
```
