# Get User ID From Name

## Description

Gets a username and outputs a user ID.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Username (String)

The username whose user ID will be retrieved.

```
ROBLOX
```

## Outputs

#### UserId (Number)

The UserId that belongs to the provided Roblox username.

```
1
```
