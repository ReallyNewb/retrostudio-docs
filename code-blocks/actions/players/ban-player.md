# Ban Player

## Description

Bans a player from the game. This prevents them from joining every server, newly created or not.

#### Available to

* [x] Scripts
* [ ] LocalScripts
* [ ] Yields

## Inputs

#### Player (Object)

The player who will be banned from playing the game.

* [x] Can use Variable

```
game.Players.ExploiterAccount
```
