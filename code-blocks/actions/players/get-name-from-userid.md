# Get Name From User ID

## Description

Gets a Roblox username from a Roblox User ID.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [x] Yields

## Inputs

#### UserId (Number)

The UserId that belongs to a Roblox user.

* [x] Can use Variable

```
1
```

## Outputs

#### Username (String)

The username that belongs to the UserId.

```
ROBLOX
```
