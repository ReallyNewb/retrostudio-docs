# Give Badge

## Description

Makes a player receive a badge.

#### Available to

* [x] Scripts
* [ ] LocalScripts
* [ ] Yields

## Inputs

#### Player (Object)

The player who will receive the badge.

* [x] Can use Variable

```
game.Players.RobloxAndTyler
```

#### BadgeId (Number)

The target BadgeId.

* [x] Can use Variable

```
1
```

## Outputs

#### Result (Bool)

Determines whether or not if Give Badge completed its operation successfully.

```
True
```
