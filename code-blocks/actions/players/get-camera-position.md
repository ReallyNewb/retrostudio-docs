# Get Camera Position

## Description

Retrieves the camera position and rotation for the local player.

#### Available to

* [ ] Scripts
* [x] LocalScripts
* [ ] Yields

## Outputs

#### Position (Vector3)

The position of the camera.

```
100, 200.4, 0
```

#### Rotation (Vector3)

The rotation of the camera.

```
90, 180, 0
```
