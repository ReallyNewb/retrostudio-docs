# Set Camera Position

## Description

Sets the camera position and rotation for the local player.

#### Available to

* [ ] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Position (Vector3)

The new position of the camera.

* [x] Can use Variable

```
100, 200.4, 0
```

#### Rotation (Vector3)

The new rotation of the camera.

* [x] Can use Variable

```
90, 180, 0
```
