# Teleport To Place

## Description

Teleports a player to another RetroStudio place.

#### Available to

* [x] Scripts
* [ ] LocalScripts
* [ ] Yields

## Inputs

#### Player (Object)

The player who will be teleported to PlaceId.

* [x] Can use Variable

```
game.Players.RobloxAndTyler
```

#### PlaceId (String)

The RetroStudio place ID that the player will be teleported to.

* [x] Can use Variable

```
3145063827_1
```
