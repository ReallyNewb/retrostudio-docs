# Register Collision Group

## Description

Registers a collision group.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### GroupName (String)

The collision group's name.

* [x] Can use Variable

```
BlacklistParts
```

#### Unregister (Bool)

Determines if the collision group should delete itself or not.

* [x] Can use Variable

```
False
```
