# Check Tool Equipped

## Description

Returns a bool based on whether or not if a tool is equipped.

#### Available to

* [x] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Tool (Object)

The tool whose equip status will be checked.

* [x] Can use Variable

```
game.Players.ExampleGuy.LinkedSword
```

## Outputs

#### Result (Bool)

Determines whether or not if the tool is equipped.

```
True
```
