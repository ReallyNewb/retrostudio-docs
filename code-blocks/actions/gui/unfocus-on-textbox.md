# Unfocus on TextBox

## Description

Makes the user stop typing in a TextBox.

#### Available to

* [ ] Scripts
* [x] LocalScripts
* [ ] Yields

## Inputs

#### Object (Object)

The target TextBox that the local player will stop focusing / typing in.

* [x] Can use Variable
